# LightMisBank
基于 Microsoft .NET 框架开发的轻量式错题管理软件。

软件主要用作练手~

## 已完成的部分
 - [x] 初始化界面
 - [x] 数据库的基本对接
 - [ ] 数据库与题库的对接
 - [ ] 题库面板
 - [ ] 设置面板

## 项目部署
直接将整个项目 Clone 至本地，使用 Visual Studio 2019+ 打开即可。

数据库文件路径定义在[Config.cs](Config.cs)内定义。
